# RISC-V architecture
## spec
* 32 + 2(MDR, MAR) register
* Little Endian
* 2's compliment
* 1 Word = 32bits
* data protocol = str
    **Delimiter( , space )** is allowed 
    - "00000000000000000000000000001111"
    - "00000000,00000000,00000000,00001111"
    - "00000000 00000000 00000000 00001111"
    **suffix (0b) is allowed**
    - "0b 00000000 00000000 00000000 00001111"
